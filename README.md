# HR-Analytics-PowerBI

## Project Overview
This project presents an HR Analytics dashboard built in Power BI to analyze employee engagement, satisfaction, work-life balance, and retention trends.

### Key Features
* Built an interactive Power BI dashboard analyzing employee engagement, satisfaction, and work-life balance trends.
* Designed and implemented a star schema data model and developed DAX measures for KPI reporting.
* Used Power Query for data transformation and cleaning.
* Identified business insights related to employee retention and departmental engagement trends.

## Dashboard Preview
See the screenshots folder for dashboard views, data model, and data transformation process.

## Data Sources
The project uses two datasets:

* Employee data
* Employee engagement survey data

Both raw and cleaned datasets are included in this repository.
